## [0.1.0] - 2026-01-23

### Added

- Initial release of Hapticlabs Player for Unity.
- Cross-platform haptic playback for Android (.hac) and iOS (.ahap).
- `HapticlabsService` singleton for managing haptic playback.
- `Custom` MonoBehaviour for triggering custom haptic files.
- `BuiltIn` MonoBehaviour for triggering built-in haptic effects.
- Sample scene demonstrating button-triggered haptics.
- Documentation and API reference.
